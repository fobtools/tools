# 小工具网站

## 产品功能
- 提供两个实用小工具
- 左侧栏显示工具列表,右侧显示工具内容
- 可以方便地在不同工具之间切换
- 抛货计算器: 根据长、宽、高和实际重量计算是否为抛货
- 运费模板计算器: 根据给定参数生成运费模板,支持人民币和美元显示

## 技术栈
- HTML5
- CSS3 (改进的UI设计)
- JavaScript
- Python (用于自动更新 README.md)
